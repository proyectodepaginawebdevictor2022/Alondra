Generalidades

Existen cuatro estados para un link, el cual es un enlace a una pá web representando para este caso con un texto

Primer Estado, original: Apariencia original del link

Para este caso los links se observación con una apariencia al iniciar la página

Segundo Estado, efecto Hover: Al pasar el mouse por56 encima del link se muestra el efecto hover

Se puede fijar la atención del usuario para que sepa si es la elección que le interesa, al retirar el mouse vuelve a su estado anterior

Tercer Estado, active: Dar Click al link

Se aprecia al clickear puede cambiar el diseño del link

Cuarto Estado visited: El link ya fue visitado
